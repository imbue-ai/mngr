# Changelog

A concise, human-friendly summary of changes. Updated nightly by the changelog consolidation agent.

For the full, unedited changelog entries, see [UNABRIDGED_CHANGELOG.md](UNABRIDGED_CHANGELOG.md).

## 2026-05-01

- Introduced a changelog system that tracks user-visible changes across PRs
- PRs now require a per-branch entry in the `changelog/` directory, enforced by CI
- A nightly agent consolidates entries into `UNABRIDGED_CHANGELOG.md` and produces concise summaries in `CHANGELOG.md`
