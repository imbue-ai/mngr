## Scope

Move minds to a multi-environment setup (`dev`, `staging`, `production`) with secrets stored in Vault instead of local `.minds/<env>/*.sh` files, and stand up isolated infrastructure per environment (Modal app, Neon DB, Cloudflare, SuperTokens, OAuth, Vultr, LiteLLM, pool-management SSH).

Production and staging should be fully isolated and identical in shape. Dev should support dynamic per-developer environments backed by a shared set of dev credentials.

Secrets pushed to Modal at deploy time stay in Modal (no way around it); the source of truth is Vault.
