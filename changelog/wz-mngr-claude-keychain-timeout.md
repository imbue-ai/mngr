- mngr_claude: bound `security find-generic-password` reads at 10 seconds so a hidden macOS keychain ACL prompt can no longer hang `mngr create` indefinitely. The symptom was Lima / remote agent creation stalling at `_setup_per_agent_config_dir` with no further log output -- the `security` child was blocked forever waiting on a dialog the user never sees or dismisses (common when the running binary's code signature doesn't match the keychain entry's ACL, e.g. after an app is re-signed). On timeout we log a clear warning pointing at Keychain Access.app + the `convert_macos_credentials=false` escape hatch, and agent creation continues without merging the keychain primaryApiKey.
