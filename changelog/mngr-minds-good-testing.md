Spec: design for live integration / acceptance / release testing of the minds app, its deployed remote services, and the deployment process itself. Introduces an operator-invoked `just minds-test-deployment` orchestrator that stands up shared dev envs, runs two parallel offload-Modal batches of pytest tests, and reliably cleans up every resource it creates via both a per-run ledger and a `ci-<timestamp>` name+age sweep. See `specs/minds-deployment-tests.md` for the full design.
