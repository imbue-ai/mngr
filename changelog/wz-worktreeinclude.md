Add `.worktreeinclude` so Claude Code worktree-isolated subagents (and `claude --worktree`) inherit the same gitignored local configs that `mngr`'s `work_dir_extra_paths` already carries: `.mngr/settings.local.toml`, `.reviewer/settings.local.json`, `.claude/settings.local.json`, `.env`, `.envrc`. Without this, agents spawned in worktree mode silently diverge from the user's local mngr/reviewer/Claude setup.
