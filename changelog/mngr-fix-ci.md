- mngr_modal: restore the autouse `_reset_modal_app_registry` fixture in `libs/mngr_modal/imbue/mngr_modal/conftest.py`. PR #1533 (`wz/mngr-modal-leak-fixes`) removed it, which caused 14 deterministic test failures on `main` because `ModalProviderBackend._app_registry` is class-level state that leaks between tests sharing a Python process. The failures were masked locally by xdist's per-worker process isolation but surfaced in offload, where each Modal sandbox runs many tests in one process.
