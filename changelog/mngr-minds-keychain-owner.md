Minds: read the macOS "Claude Code" keychain entry in the Electron main process at backend startup and forward the value to `mngr create` as `ANTHROPIC_API_KEY` via the subprocess env. On macOS, this means any keychain ACL prompt that does appear is attributed to minds.app's signing cert (so "Always Allow" survives ToDesktop rebuilds), is parented to the minds.app window at a known UX moment instead of during a later `mngr create`, and one prompt covers the lifetime of the install instead of one-per-`security`-shellout. `mngr create` from the terminal is unchanged.
