`mngr create --type X` now fails fast with `UnknownAgentTypeError` when `X` is not a registered agent type and not defined under `[agent_types.X]` in config, instead of silently resolving to a generic `BaseAgent` + empty config. Use `--type command -- <shell command>` to run an arbitrary shell command. The `--type X -- ...` form is no longer a hidden alias for `--type command -- ...`.
