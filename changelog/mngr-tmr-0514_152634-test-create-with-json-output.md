- Fix `test_create_with_json_output` e2e test by adding `@pytest.mark.timeout(120)`. Without an explicit timeout, the test was subject to the global 10s pytest-timeout, which is insufficient for `mngr create` plus the follow-up `mngr list` calls. The test now also verifies that the JSON output of `mngr create --format json` contains the expected `agent_id` and `host_id` fields and that the listed agent matches the one that was created.
