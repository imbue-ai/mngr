- Changed: `mngr pair` source flags simplified to a single `HostedLocation`. The positional `SOURCE` and `--source` remain (both accept `AGENT[@HOST[.PROVIDER]][:PATH]`); the redundant `--source-agent`, `--source-host`, and `--source-path` are removed. `--source-host` was previously silently ignored when used without an agent in interactive mode. To pin an agent to a specific host, use `AGENT@HOST` syntax (e.g. `mngr pair my-agent@localhost`). Specifying only a host (e.g. `mngr pair @localhost` or `mngr pair @localhost:/path`) is rejected with a clear error since pair syncs through an agent.
