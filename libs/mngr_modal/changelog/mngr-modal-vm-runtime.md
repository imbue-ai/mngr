Added an `is_vm_runtime_enabled` setting to the Modal provider config that controls whether sandboxes run on Modal's VM runtime (https://modal.com/docs/guide/vm-sandboxes) instead of the default gVisor runtime. It defaults to true, so newly created and snapshot-restored sandboxes now use the VM runtime; set `is_vm_runtime_enabled = false` under `[providers.modal]` to opt back out.
