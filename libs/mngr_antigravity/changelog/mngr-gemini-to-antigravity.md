Rename `mngr_gemini` to `mngr_antigravity`; agent type `gemini` is replaced by `antigravity`. Google announced on 2026-05-19 that the Gemini CLI is being superseded by the Antigravity CLI (`agy`); the legacy request path turns off for paid-tier accounts on 2026-06-18. The plugin was never released, so this is a destructive rename with no shim. The new CLI is architecturally closer to Claude Code than to Gemini CLI: process name is `agy`, hook event names match Claude's (`SessionStart`, `PreToolUse`, `PostToolUse`, `SessionEnd`, `Stop`, `Notification`), and `--dangerously-skip-permissions` is the documented auto-allow flag. `auto_allow_permissions=True` is now wired through that CLI flag rather than a permission hook. The readiness sentinel and `mngr transcript` support carried by `mngr_gemini` have been dropped in v0 -- the Antigravity hook schema and `.pb` conversation format could not be empirically validated against an authenticated CLI in this PR, and the plan's drop-if-blocked permission applies. Both are tracked as follow-up work.
