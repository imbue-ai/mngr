Added `docker_runtime` and `install_gvisor_runtime` options to the VPS docker provider config. `docker_runtime` (default unset) passes `--runtime=<value>` to the container's `docker run` on the VPS. `install_gvisor_runtime` (default false) makes VPS provisioning install and register the gVisor `runsc` runtime with the Docker daemon via gVisor's official APT repository; the step is idempotent and a no-op when runsc is already present (e.g. baked into the image). Installing is independent of enabling -- set `docker_runtime = "runsc"` to actually run containers under gVisor.
