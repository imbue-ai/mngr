Added `OfflineCapableVpsDockerProvider`, a base for cloud providers (AWS/GCP/Azure) whose hosts can be stopped while their disk persists. It consolidates the previously-duplicated offline discovery and host resolution -- reconstructing stopped (SSH-unreachable) hosts and their agents from the provider's instance listing, and falling back to that listing when the on-volume path is unreadable -- behind a small set of per-provider hooks. No user-visible behavior change.
