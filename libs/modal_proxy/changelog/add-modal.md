Added `RemoteModalInterface`, a `ModalInterface` implementation for `ModalMode.PROXIED` that proxies the testing subset of sandbox operations (create, exec, tunnels, get/set tags, list, terminate) to the imbue_cloud connector over HTTP, so the client never holds Modal credentials. Persistent-host features (volumes, snapshots, image builds, deploy) are unsupported in this mode and raise a clear error. Also adds the `/modal/sandboxes` wire-protocol request/response models shared with the connector.
