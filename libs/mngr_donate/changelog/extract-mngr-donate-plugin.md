New `imbue-mngr-donate` plugin. `mngr donate` (and its launchd `--start`/`--stop` scheduler) moved out of `imbue-mngr-usage` into this standalone plugin, since donating spare quota is orthogonal to measuring usage. The plugin depends on `imbue-mngr-usage` (installing `imbue-mngr-donate` pulls it in) and reads the `usage` plugin's spare-capacity snapshot at runtime. `mngr donate` now also pulls the donation skill's prompts fresh from the skill's upstream git repo at run time (the `--skill-repo` option) so experiment prompts can be updated without an mngr release, while the skill's pinned code stays reviewed in this repo. The pinned `document-review` client no longer supports the `ANTHROPIC_API_KEY` (`run`/`auto`) path, so donation runs can only ever spend Claude subscription quota, never direct API billing.
