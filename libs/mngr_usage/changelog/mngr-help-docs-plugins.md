The usage plugin now contributes its cron-recipes documentation as a `mngr help` topic via mngr's new `register_help_topics` hook. With the plugin installed, `mngr help` lists `usage_cron_recipes` ("mngr usage: Cron automation recipes") and `mngr help usage_cron_recipes` shows the cron automation recipes. The topic key and description are namespaced so they are unambiguous in the global topic list.
