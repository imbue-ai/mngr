- Fixed the release e2e test `test_destroy_remove_created_branch_inline` (in `libs/mngr/imbue/mngr/e2e/tutorial/test_git.py`), which covers `mngr destroy my-task --force --remove-created-branch`. Added `@pytest.mark.timeout(60)` (the create+destroy sequence exceeds the default 10s function timeout) and dropped the superfluous `@pytest.mark.modal` mark, since the agent is created and destroyed on the local provider and destroy's garbage collection never reaches Modal (the Modal environment is never created, so the provider is skipped before any Modal call). The test now also asserts the concrete effects of the command: the agent is gone from `mngr list` and the created `mngr/my-task` git branch no longer exists.
