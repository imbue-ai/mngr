- Removed the superfluous `@pytest.mark.modal` from the `test_list_format_jsonl_recap` e2e test. A read-only `mngr list` in a fresh test environment never invokes the Modal CLI binary (the only Modal usage the resource guard can observe across the `mngr` subprocess), so the mark tripped the guard's "marked but never invoked" check and failed the test. The test now also parses the `mngr list --format jsonl` output and asserts every emitted line is a standalone JSON object, verifying the JSONL "one object per line" contract instead of only the exit code.
