- Fixed the `test_plugin_enable_project_scope` e2e tutorial test. It previously asserted that `mngr plugin enable my-plugin --scope project` either fails or reports "not found", but `plugin enable` is a soft operation that succeeds (with a warning) and records the enabled state in the project config so a plugin can be pre-configured before installation. The test now verifies that the command succeeds, warns that the plugin is not currently registered, and actually writes `plugins.my-plugin.enabled = true` into the reported project-scope `settings.toml`. Also added `@pytest.mark.timeout(60)` so the test is not killed by the default 10s per-test timeout, which is shorter than mngr's cold-start time.
