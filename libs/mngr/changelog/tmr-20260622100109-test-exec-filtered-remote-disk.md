Fixed the `test_exec_filtered_remote_disk` release test, which exercises filtering agents with `mngr list --include 'host.provider == "modal"'` and piping the result into `mngr exec`. The exec step runs over a freshly established SSH connection to a remote Modal host, which exceeds the default 30s command timeout on a cold connection. Gave the exec command an explicit 120s timeout and raised the per-test timeout to 300s to accommodate the create plus exec.
