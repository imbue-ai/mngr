Test-quality improvements across the agents test suite (`libs/mngr/imbue/mngr/agents`). No production behavior changed; these strengthen weak, tautological, or implementation-coupled tests so they catch real regressions.

- `test_base_agent.py`: pinned `get_command` to its data.json source, tightened `record_activity` assertions (timestamp window, per-source isolation, JSON record), added a `HostConnectionError -> STOPPED` lifecycle test and a real-tmux RUNNING test, added the `cli_args` splice ordering case, added positive coverage for `get_reported_url`, `list_reported_plugin_files`, and `runtime_seconds`, converted representative setter round-trips to read back through a freshly constructed agent (proving on-disk persistence), removed a mislocated unit test, and stopped reaching into private data-path internals.
- `base_agent_test.py`: short/long message send tests now assert the actual message, target pane, and consistent tmux buffer name (not just loose substrings); the inline host stub's modeled methods now mirror the real host signatures; the `as_shell_arg` check asserts documented properties instead of a hand-built string; the runtime_seconds and activity-delta bounds were tightened; added a direct unit test of `_get_command_basename` covering the subshell/empty branches; representative setters now verify on-disk persistence via a fresh instance.
- `tui_agent_test.py`: replaced structural/tautological assertions with behavior tests that drive the real `send_message`/`wait_for_ready_signal` pipelines against an in-memory probe.
- `base_headless_agent_test.py`: dropped a redundant private-method test, and made the stopped-with-pane-content probe carry `pane_content` as a per-instance field instead of a shared mutable class attribute.
- `default_plugins/headless_command_agent_test.py`: deleted a tautological `--print` assertion, spawned the tmux session through the host interface (not a raw subprocess) with an explicit session-name linkage assertion, and documented the load-bearing `local_provider` fixture.
- `tui_utils_test.py` and `agent_registry_test.py`: clarifying comments documenting the minimal recorder host and the default-binary-name contract.
