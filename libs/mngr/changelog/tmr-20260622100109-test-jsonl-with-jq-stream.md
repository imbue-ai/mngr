Fixed the `test_jsonl_with_jq_stream` e2e tutorial test for the `mngr list --format jsonl | jq` streaming example. The test was timing out under the default 10s cap (Modal discovery is slow) and was incorrectly marked `@pytest.mark.modal`, which would fail the resource guard's NEVER_INVOKED check because the read-only `mngr list` path never shells out to the `modal` CLI. Removed the `modal` mark, raised the per-test timeout to 180s, and strengthened the assertions to verify the jq pipeline produces empty stdout (no RUNNING agents) rather than only exiting 0.
