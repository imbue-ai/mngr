Fix the "run a command on all agents" example in the mega tutorial. It previously showed `mngr exec -a "whoami"`, but the `-a`/`--all` flag was removed from the multi-target commands; the documented way to fan out across every agent is now to pipe `mngr list --ids` into `mngr exec -`. The tutorial now reads `mngr list --ids | mngr exec - "whoami"`, matching the rest of the tutorial's multi-agent examples.
