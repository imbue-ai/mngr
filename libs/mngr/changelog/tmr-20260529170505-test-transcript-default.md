- Fixed the `test_transcript_default` e2e tutorial test. It previously created a `--type command` sleep agent and ran `mngr transcript my-task`, which always failed because command agents do not produce a common transcript (the command exits non-zero with "does not produce a common transcript"). The test now seeds a `claude`-type agent with a sample common transcript (via `imbue.mngr.cli.testing.create_agent_with_sample_transcript`) so the tutorial command has real conversation data, and it asserts on the rendered output (user/assistant messages) rather than just the exit code. The `tmux`/`rsync`/`modal` resource marks were dropped since the agent is local and `mngr transcript` resolves it without contacting those resources.
