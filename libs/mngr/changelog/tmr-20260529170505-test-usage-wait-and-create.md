- The release-marked e2e tutorial test `test_usage_wait_and_create` now asserts the actual behavior of the `mngr usage wait ... && mngr create` composition: the wait times out (exit code 2) within its `--timeout`, and the chained create is short-circuited (verified via `mngr list --format json` showing no `chore` agent). The superfluous `@pytest.mark.modal` mark was removed (the `--timeout`-driven path never creates a modal agent), and a `@pytest.mark.timeout(120)` guard was added.
