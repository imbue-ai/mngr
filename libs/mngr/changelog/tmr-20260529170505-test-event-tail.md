- Fixed the `test_event_tail` e2e tutorial test (`libs/mngr/imbue/mngr/e2e/tutorial/test_event.py`): it creates a local command-type agent, so it never invokes Modal and was tripping the resource guard for its `@pytest.mark.modal` mark; removed that mark. It was also hitting the default 10s pytest timeout while creating the agent (creation takes ~26s), so added `@pytest.mark.timeout(60)`.
- Strengthened `test_event_tail`: beyond asserting the command succeeds, it now parses the `mngr event --tail 20` output as JSONL, checks that `--tail 20` never returns more than 20 events, and verifies each emitted event carries the documented `event_id`/`timestamp`/`source`/`type` fields.
- Added `test_event_tail_rejects_combination_with_head`, an unhappy-path companion that verifies `mngr event --tail` cannot be combined with `--head` (exits non-zero with "Cannot specify both --head and --tail").
