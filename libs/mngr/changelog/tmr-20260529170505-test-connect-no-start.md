- Fixed the release-marked e2e test `test_connect_no_start` in `libs/mngr/imbue/mngr/e2e/tutorial/test_connect.py`. It now exercises the documented "(fails if agent is stopped)" behavior of `mngr connect --no-start`: it stops the agent, asserts that `mngr connect my-task --no-start` exits non-zero with a "stopped" error instead of auto-starting, and confirms via `mngr list --stopped` that the agent was not started as a side effect. Previously the test attempted an interactive `tmux attach` (which has no terminal under the e2e runner) and lacked a `@pytest.mark.timeout`, so it hit the 10s default pytest timeout and then failed with `open terminal failed: not a terminal`. Added a `@pytest.mark.timeout(300)` marker and removed the inapplicable `@pytest.mark.modal` mark (the local-only flow never reaches the guarded Modal chokepoint).
