- Fixed the e2e tutorial test fixture (`libs/mngr/imbue/mngr/e2e/conftest.py`): the isolated test profile now configures a default agent type (`commands.create.type = "claude"`), mirroring an onboarded user, so tutorial `mngr create` commands that omit `--type` no longer fail with "No agent type provided" now that `mngr create` no longer hard-codes a `claude` default. - `test_create_modal_idle_timeout_60` now verifies the actual effect of `--idle-timeout 60` by listing the created Modal agent and asserting `idle_timeout_seconds == 60`, rather than only checking the command's exit code.
