- Fixed the e2e release test `test_exec_as_other_user` in `libs/mngr/imbue/mngr/e2e/tutorial/test_exec.py`. It was failing for two reasons: it inherited the default 10s `pytest-timeout` (too short for the real `mngr create` it performs), and it carried a superfluous `@pytest.mark.modal` even though it only creates and execs against a local command agent (the resource guard fails marked-but-never-invoked tests). Added `@pytest.mark.timeout(300)` and removed `@pytest.mark.modal`.
- Strengthened the test to assert on the actual exec output (the host uid) rather than only the exit code, so it verifies the quoted command really ran on the agent host with the expected user context.
