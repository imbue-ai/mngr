- Fixed the `test_gc_default` e2e tutorial test (`libs/mngr/imbue/mngr/e2e/tutorial/test_gc.py`). It previously ran `mngr gc` against an empty environment and failed the `@pytest.mark.modal` resource guard with "marked with @pytest.mark.modal but never invoked modal": a bare `mngr gc` only reaches Modal through the in-process SDK, which the guard cannot observe across the subprocess boundary. The test now first creates a live Modal command agent (which invokes the `modal` CLI during environment creation, satisfying the guard and giving gc real resources to scan), then runs `mngr gc`, and asserts that the live agent survives -- verifying that gc only collects *unused* resources.
