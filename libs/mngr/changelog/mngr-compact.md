Fixed `mngr message` timing out after 90 seconds for every message sent to an idle agent on macOS (and any other host without GNU coreutils). The tmux submission waiter was bounded with `timeout`, which macOS does not ship; the command failed with exit 127, its error was swallowed, and the waiter was never registered, so the `UserPromptSubmit` hook's `tmux wait-for -S` signal woke nobody. The waiter is now bounded with plain bash, which these commands already run under. On the affected host, a normal prompt now confirms in 2.1 seconds instead of failing after 93.6 seconds; `/clear` does too.

Fixed a latent false-success in the same path. tmux remembers a `wait-for -S` sent while no waiter is registered, so the next `wait-for` on that channel returns immediately. A signal left behind by an earlier submission would have confirmed the next `mngr message` before its Enter was even processed. The channel is now drained before the waiter is registered. Two related hazards are also handled: the deadline is enforced with `SIGKILL`, because `tmux wait-for` handles `SIGTERM` and exits 0 (indistinguishable from being woken), and the waiter is never wrapped in a subshell, because killing the subshell would orphan a `tmux wait-for` client that remains registered on the channel and steals a later submission's signal.

Fixed the raw-transcript streamer's offset reconciliation, which used `tac` (GNU-only). On macOS the resulting `tac: command not found` was printed into the agent's terminal, painting over Claude's input row; the next `mngr message` then failed after 16.7 seconds when its paste-visibility check could not find the pasted text. Reversal now prefers `tac` and falls back to `tail -r`.

Added `specs/mngr-message-delivery-signals.md`, which catalogs the input categories `mngr message` must handle, inventories every signal that Claude Code and mngr emit (with the component that produces each), maps signals to cases in a measured matrix, and records the remaining known issues. Added a top-level `uncertainties.md` noting two comments that assert behavior which does not hold.
