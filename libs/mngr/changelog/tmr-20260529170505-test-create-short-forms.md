- Removed the superfluous `@pytest.mark.modal` mark from the `test_create_short_forms` e2e tutorial test. The test only creates local agents and never invokes Modal through a guard-tracked path (`mngr list`'s Modal discovery uses the Modal SDK inside the `mngr` subprocess, where the in-process SDK guard monkeypatch does not apply, and no `modal` CLI binary is invoked), so the resource-guard `NEVER_INVOKED` check would always fail. Also strengthened the test's assertions to verify that both the full (`mngr create <name> <type>`) and short (`mngr c <name>`) forms produce equivalent, actually-running agents in worktrees.
