Added an OVH catalog pricing helper (`pricing.compute_order_pricing`) that computes the true all-in month-to-month cost of a dedicated-server order: base plan plus every selected add-on delta (RAM/storage/bandwidth), the one-time setup fee, and the first payment. This is the shared, tested source of truth that the upcoming bare-metal admin ordering flow will quote from, so the catalog's bare "base" price can no longer be mistaken for the real recurring cost.
