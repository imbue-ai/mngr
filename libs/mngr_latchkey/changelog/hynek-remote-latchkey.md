Added `libs/mngr_latchkey/imbue/mngr_latchkey/remote_gateway.py`, the first piece of "run the latchkey gateway on the VPS" support. It declares a pinned `LATCHKEY_VERSION` (2.15.1) and an `ensure_latchkey_installed(host)` function that provisions the upstream `latchkey` CLI and its runtime prerequisites (curl, Node.js via NodeSource, and `latchkey@<version>` via `npm install -g`) on a remote VPS (the agent's outer host). The install is idempotent: each component is installed only when missing, or, for latchkey, when the installed version differs from the pinned version. It runs as a single POSIX-sh command (no bash-only `pipefail`), with a generous install timeout and a slow-install warning.
