# Service icons

Monochrome brand icons for the creating-page onboarding carousel, keyed by
latchkey canonical service name (`<service_id>.svg`).

Source: the [simple-icons](https://github.com/simple-icons/simple-icons)
project (CC0-1.0), copied verbatim from the npm package (v13.21.0). Services
with no simple-icons entry (e.g. `coolify`, `ramp`) have no file here; the
carousel renders a monogram fallback for them.

The SVGs have no `fill` attribute, so they inherit `currentColor` from the
surrounding element.
