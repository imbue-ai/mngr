Routed `apps/minds/scripts/build_test.py` to a Node-equipped CI job. PR #1772 renamed `apps/minds/todesktop.json` to `apps/minds/todesktop.js`, so the test that evaluates that config (`test_bundled_limactl_is_signed_with_virtualization_entitlement`) now needs a Node runtime on PATH. The test is marked with the repo-wide `node_required` pytest marker (defined in `libs/imbue_common`) and runs in the dedicated `test-node` CI job, replacing the previous `skipif(node is None)` guard. A missing Node binary is now a hard failure surfacing a real misconfiguration rather than a silent skip.
