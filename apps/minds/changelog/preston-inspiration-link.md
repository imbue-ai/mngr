Inspiration deeplinks (`minds://create?git_url=...`) now land on a dedicated Create from Inspiration page instead of the plain create form. The page first asks whether to create a new workspace from the Inspiration or add it to an existing one.

Adding to an existing workspace is a two-step walkthrough: copy the `/use-inspiration <repo>` message, then pick one of your workspaces to open and paste it into that workspace's chat (stopped local workspaces are restarted on the way in, matching the landing page).

Creating a new workspace shows only the cloud-vs-local choice -- no advanced settings -- with the Inspiration repository as a read-only confirmation and a required "I trust this Inspiration" acknowledgment (Inspirations are community content, not approved or verified by Imbue). The create button reads "Create from Inspiration"; the workspace is otherwise configured exactly like a normal preset create.
