Inspiration deeplinks (`minds://create?git_url=...`) now land on a dedicated Create from Inspiration page instead of the plain create form. The page is a numbered vertical timeline that reveals each step only once the previous one is done. Step 1 asks whether to create a new workspace from the Inspiration or add it to an existing one.

Adding to an existing workspace walks through: (2) copy the `/use-inspiration <repo>` message -- the workspace picker stays hidden until you copy -- then (3) pick one of your workspaces to open and paste it into that workspace's chat (stopped local workspaces are restarted on the way in, matching the landing page).

Creating a new workspace walks through: (2) cloud vs. local -- no advanced settings -- then (3) the remaining confirmation: the account, the Inspiration repository shown as plain read-only text, and a required "I trust this Inspiration" acknowledgment (Inspirations are community content, not approved or verified by Imbue). The create button reads "Create from Inspiration"; the workspace is otherwise configured exactly like a normal preset create.
