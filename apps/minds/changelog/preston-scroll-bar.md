The document scrollbar on scrolling local pages (Settings, Home, ...) is now contained below the fixed titlebar instead of running the full window edge over it: the scrollbar track carries a 38px top margin so the thumb can never ride over the bar, and the strip's top notch paints in the titlebar's own color (accent or neutral) so the bar reads edge-to-edge. When a page cannot scroll -- including right after swapping away from a scrolling page -- no thumb paints and the track matches the page surface, so the scrollbar visibly disappears with the page that owned it.
