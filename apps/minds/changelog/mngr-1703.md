- The minds desktop client no longer passes `--reuse` / `--update` when creating workspace agents. Every minds launch mode (LOCAL, LIMA, CLOUD, IMBUE_CLOUD) passes `--new-host`, which `mngr create` now rejects in combination with `--reuse`. The IMBUE_CLOUD pre-baked `system-services` agent is adopted via mngr's `host.pre_baked_agent_id` duplicate-name exemption, so `--reuse` was unnecessary.
