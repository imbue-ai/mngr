Added "Modal (experimental)" as a first-class compute provider in the Create workspace form, alongside Docker / Lima / Vultr / AWS / Imbue Cloud. Selecting it runs the agent in a Modal sandbox provisioned keyless through the imbue_cloud connector (`ModalMode.PROXIED`) -- no local Modal token needed. The single `[providers.modal]` block (PROXIED mode, pointing at the same connector URL as imbue_cloud) is registered at startup whenever the user has an active gateway session. Modal sandboxes are ephemeral (~1 day max), so the option is labeled experimental/testing-only in the create dropdown and the workspace listing.
