Simplified the minds release runbook (`apps/minds/docs/release.md`): the `forever-claude-template` `vendor/mngr` refresh no longer goes through a PR. It is a mechanical, reproduction-verified mirror of the tagged mngr SHA, so it now lands as a direct commit pushed straight to FCT `main` (no review). The mngr side still releases via a PR. Reviewable FCT consumer changes (e.g. a `system_interface` fix) remain a separate, normal FCT PR landed before the vendor push.
