# minds agent instructions

- default-workspace-template (shorthand **dwt**) is the workspace-template repo minds provisions agents from, and by far the most common external repo you develop alongside mngr. To develop it from this mngr checkout, run `just default-workspace-template-worktree` (short alias `just dwt-worktree`): it creates an independent default-workspace-template checkout at `.external_worktrees/default-workspace-template` on the current mngr branch, needing no configuration (it clones the template directly; errors if a checkout is already there). Commit template changes inside that checkout and open a separate PR against default-workspace-template. If the user already has a local default-workspace-template clone and `DEFAULT_WORKSPACE_TEMPLATE_DIR` is unset, suggest they add `DEFAULT_WORKSPACE_TEMPLATE_DIR=<path>` to a gitignored `apps/minds/.env` so `just default-workspace-template-worktree` (and `bake-slice-dev`) reuse it instead of a fresh clone.
