Add the open-seer app: an autonomous Sentry-error-to-PR system. It polls Sentry for new issues, triages them, and drives Claude Code sessions (via the fix-sentry-error and sentry-sweep skills) to produce candidate fix PRs. Includes the tick loop, a three-scanner secrets/PII gate (Betterleaks with custom PII rules, TruffleHog, and Kingfisher — all must pass before a fixer's PR goes up), Dockerfile, design docs, and tests.
