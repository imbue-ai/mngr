Added a design spec (`specs/common-transcript-standard/spec.md`) proposing that the agent-agnostic common-transcript schema track the OpenTelemetry GenAI semantic conventions instead of bespoke field names. It scopes a Tier 1 vocabulary alignment (e.g. `stop_reason` -> `finish_reason`, tool-call field renames) across all five emitters, and an additive, optional ordered `parts[]` field (Tier 2) for the agents whose native format preserves text/tool-call interleaving (claude, pi-coding), with the parity gap documented for codex (text-only messages), opencode (deferred), and antigravity (not reconstructable from its native format).
