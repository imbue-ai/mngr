Added a dedicated `test-node` CI job (`.github/workflows/ci.yml`) that runs all Python tests marked `node_required` on a lightweight runner provisioning only Python, uv, and Node.js (no pnpm / Electron deps / xvfb / Docker). Excluded the `node_required` marker from every offload run (`offload-modal.toml`, both the `all` and `flaky` groups) and from the local `just test-unit` / `test-integration` / `test-quick` filter (`justfile`), since neither the shared mngr Modal image nor a typical dev machine has Node. This replaces the earlier approach of piggybacking the test on the heavier `test-docker-electron` job.
