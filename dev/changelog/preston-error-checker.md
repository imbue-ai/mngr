Added `specs/window-error-watcher/`: spec and 5-task implementation plan for a new forever-claude-template (FCT) background service that scans every tmux window in the agent session for `/error|exception/i` output and, on newly-appeared matches, messages a random mngr agent. The implementation itself lives in the FCT repo (`imbue-ai/forever-claude-template`, branch `preston/error-checker`), not in this monorepo; this entry covers only the spec + plan added under `specs/`.

Fixed `just minds-start`: its `vendor/mngr/` rsync relied on the `.gitignore` rule `**/.external_worktrees/` to skip the worktree dir, but macOS's stock rsync (openrsync, "2.6.9 compatible") does not support `**` globs, so it copied `.external_worktrees/` (which contains the rsync destination) into itself and raced its own `--delete`, failing with exit 23. Added an explicit `--exclude=.external_worktrees`, which works on both old and new rsync.
